<h2>Django PhoneBook v.1.1</h2>

<h3>Requirements</h3>
    <li>Python3 (3.6-3.9)</li>
    <li>MySql (MariaDB)</li>
    
<h3>Installation</h3>
 <li>python -m pip --upgrade pip</li>
 <li>python -m pip install pipenv</li>  
 <li>pipenv install -r requirements.txt</li>

<h3>How to use</h3>
<li>python manage.py runserver:8000</li>

<h3>Tested on</h3>
    <li>Manjaro</li>
    <li>Debian</li>
    <li>Fedora</li>
